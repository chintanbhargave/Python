This program takes stock data of the specific company of users choice from user chosen start and end date using pandas_datareader library. Using this data a candlestick chart is generated using bokeh. Green candles will represent the dates on which closing value of stock market is greater than the opening value and red candles will represent when closing value of stock market is less than the opening value. Line segment will represent the highest and lowest value on that day.

Libraries used in this program are 
    pandas_datareader
    datetime
    bokeh
